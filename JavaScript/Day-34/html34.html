Types of Coercion

There are two type of coercion-:

1> Implicit Coerciosn- In this javascript automatically convert one type of data to another using built in function such as Boolean(),Number().Example-

2> Explicit Coercion- In this when user changes one data type to another data type then this type of coercion is called Explicit Coercion. Example- let number = Number("35");
